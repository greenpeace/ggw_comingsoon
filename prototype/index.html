Prototype overview

The prototype consists of an anonymous section and a member section. Community members have some more options available than normal users. You can switch between both sections using the login page and log out button, which just act as demo links (no validation). Pages for comminuty coaches usually have incorrect urls, so use the browser back button if you get a page not found (sorry).

The following pages are available in the prototype. You can use links on the pages itself, most should work.

About the prototype

  • This prototype can be used as a style and code reference;
  • It does not process any forms nor does it hold or displays user generated info;
  • All content is purely meant as demo content;
  • Some interactions and feedback is provided with javascript, using merely static resources;
  • Autocompletes and suggestion lists are static, they do not respond to your input.
