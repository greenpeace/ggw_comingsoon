Collapse

Collapse can be used to toggle content. Primary use is now for the FAQ page where we show the answer for a question, by clicking on the question itself. This pattern can be extended or reused.

The href attribute in this component is ignored. Javascript will always execute on clicking the .node-title element inside a node-collapse. Then it will look for the next sibling .field-name-body and show this block.
